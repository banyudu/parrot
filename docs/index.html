Parrot

On-device dictation for macOS. Speak naturally, get clean text — no cloud, no server, powered by MLX on Apple Silicon.

Download DMG View on GitHub

100% Private

All processing happens on your Mac. No audio or text ever leaves your device.

Real-Time Streaming

See transcription appear word-by-word as you speak, with live text updates.

LLM Polish

A local language model cleans up punctuation, filler words, and dictation artifacts.

Multiple Models

Choose from Qwen3-ASR, Parakeet-TDT, or Voxtral-Mini — different sizes and languages.

Multilingual

Supports English, Chinese, and auto-detection. Switch languages from the menu bar.

Lightweight

Lives in the menu bar. Models auto-offload after idle to reclaim RAM.

Quick Start

  • Download and install from the DMG
  • Grant Microphone and Accessibility permissions
  • Hold Right Command and speak
  • Release — transcribed text is typed into the active app

Build from Source

git clone https://github.com/banyudu/parrot.git
cd parrot
make build SIGN_IDENTITY=-
make install

Requires macOS 14+, Apple Silicon, and Xcode 26.3+ with Swift 6.2.
